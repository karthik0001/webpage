Download the ForgeRock Authenticator application from Android Store / Apple Store etc.

Scan the barcode for the first time to sync the OTP. Included in instructions.

FYI: To login as an individual user, use the credentails- username=user.X and password=user.Xuser.X
Example: UserName=user.23 and Password=user.23user.23

Cronjob resets the passwords if changed, value of X ranges from 0 to 999.

Overview.

Live Demo Here Two-factor Authentication

We will cover following:

  1. Create a new realm called /2fa. Its better to create a new realm so we can configure authorization policies which are specific for this module. It would be easier to trace logs / auditability. Authentication cookie issued here won't effect other modules.
  2. Create a set of module that would be required for registration, ldap authentication, Adaptice risk, SecurID etc. as many as we can. We will not use the modules directly. We using Chains
  3. Create an Authentication Chain, combine required auth modules by setting up pass/failure rules.
  4. Change the default organization authentication to the created chain.
  5. Setup can be testing by going to the /2fa realm that was created in Step 1
