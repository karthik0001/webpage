Using OpenDJ LDAP user directory store as Identity Provider and federating users to AWS console access, service provider.

Using OpenAM, OpenDJ, OpenIG, IAM policies in AWS console, Access tokens.

FYI: To login as an individual user, use the credentails- username=user.X and password=user.Xuser.X
Example: UserName=user.23 and Password=user.23user.23

Cronjob resets the passwords if changed, value of X ranges from 0 to 999.
